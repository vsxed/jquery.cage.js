jquery.cage.js

This is a jQuery Plugin created to "cage" a website.
Everybody loves Nicolas Cage

This plugin requires jQuery to work.
It uses images from PlaceCage.

What does this plugin do?

This plugin replaces any elements background with some random-sized pictures of Nicolas Cage.
You can specify which element will be caged.

How to use it?

You can use it by simply calling this code in your html/js

$('body').cage();

Your target can be any DOM element. Choose wisely.


See example markups in the archive or on Github.

Options

There also some options you can set.
The default settings are:

type: 'calm',
auto: false,
refresh: 1500

type
You can choose the type of image which will be set.
Here are the values you can set:

Value Description
calm Default value, will return calm portraits of Mr. Cage
crazy CRAZY pictures! You will recognize some, just like the "You don't say" picture
grey Some greyscaled pictures. Less disturbing background than the other ones

auto
With auto you can turn on automatic reloading of the background picture

Value Description
true Pictures will be reloaded automatically
false Static picture. Will only reload by reloading the page

refresh
refresh sets the time to refresh the images - in milliseconds.
It will be only used, if auto is set to true

Value Description
3000 Default value (3 seconds)
any value Use any time in milliseconds
